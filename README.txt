Задача «Docstrings2pdf»
Версия 1.0
Автор: Слышева Александра
Описание
Данное приложение переводит извлекает данные о структуре програмы и docstrings на языке python
и представляет их в формате pdf.
Состав
•	Запускаемый модуль: docstring2pdf.py
•	Модуль перевода кода во внутренний формат: code_parser.py
•	Модуль перевода кода в формат pdf: formatter.py
Консольная версия
Справка по запуску: ./ docstring2pdf.py --help
Справка по командам: help [команда]
Подробности реализации
Резулбтат работы программы сохраняется в текущей директории в папке results
